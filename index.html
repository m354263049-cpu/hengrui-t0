策略说明
📌 做T是什么
持有底仓，利用日内波动高抛低吸，降低成本。A股T+1需提前持有底仓才能卖出。

📊 7大信号因子
  1. 布林带回归 — 触及下轨做多，触及上轨做空
  2. 均线支撑 — 回踩MA20获支撑做多
  3. MACD信号 — DIF/DEA金叉做多，死叉做空
  4. KDJ拐头 — K值超买超卖区拐头
  5. 缺口回补 — 跳空缺口回补动力
  6. 反转形态 — 连涨连跌后反转
  7. 量能配合 — 缩量/放量判断多空

⚡ 评分规则
每因子 +1(多)/-1(空)/0(中)，总分 -7~+7
≥+2.5 → 做多 | ≤-2.5 → 做空 | 其他 → 观望

⚠️ 风险提示
本工具基于公开技术指标，数据来自腾讯/东财等公开接口，仅供学习参考，不构成投资建议。
操作建议
做多做T：
  1. 底仓已持有
  2. 信号显示做多
  3. 支撑位附近买入
  4. 反弹至压力位卖出

做空做T：
  1. 信号显示做空
  2. 先卖出可卖部分
  3. 回落至支撑位买回
  4. 注意T+1当日买入不可卖

仓位建议：单次做T不超过底仓1/3。
